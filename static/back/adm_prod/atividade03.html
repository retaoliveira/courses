Fazer-para-estoque é um tipo de planejamento em que algumas operações produzem bens ou serviços para estoque com antecedência em relação a qualquer pedido firme. Ex: alguns construtores vão construir casas ou apartamentos padrão pré-projetados antes de qualquer demandafirme. Nos sistemas de produção que vimos anteriormente, este se encaixa em: a) Sistema puxadob) Sistema just in timec) Sistema empurradod) Sistema Kanban11. Fazer-contra-pedido (make-to-order) - Um construtor de casas que tenha padronizado projetos pode optar por construir cada casa somente quando um consumidor tenha colocado um pedido firme. Devido ao fato de o projeto da casa ser relativamente padronizado, os fornecedores de materiais terão sido identificados, mesmo que a operação de construção não mantenha os itens em si em estoque. Este é um tipo de sistema de produção: a) Sistema puxadob) Sistema just in timec) Sistema empurradod) Sistema Kanban
